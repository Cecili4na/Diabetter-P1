# Front P1 - Aplicativo Flutter

Aplicativo mobile desenvolvido em Flutter.

## Pré-requisitos

- Flutter SDK (versão 3.0.0 ou superior)
- Dart SDK (incluído com Flutter)
- Android Studio / Xcode (para desenvolvimento mobile)

## Instalação

1. Instale o Flutter seguindo as instruções em: https://flutter.dev/docs/get-started/install

2. Clone ou navegue até este diretório

3. Instale as dependências:
```bash
flutter pub get
```

## Executando o App

Para executar o app em um dispositivo ou emulador:

```bash
flutter run
```

Para listar dispositivos disponíveis:
```bash
flutter devices
```

## Estrutura do Projeto

```
lib/
  main.dart          # Arquivo principal da aplicação
pubspec.yaml         # Configurações e dependências do projeto
```
